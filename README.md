# NewJK Royal International

### Getting Started

```
 Install node_modules via 'npm install'  
 App.vue is start of application
 Go login if user is not logged in
 Home.vue is start of application for router view
 WholeSysm.vue is for show  all table state data
 HjSystem.vue is for showing single Desk Information , and Member Information
 SmmeryData.vue is for showing all total information and summery data information
 BetCompare.vue is for Comparing bet of user
 ```

```sh
npm i
npm run dev
npm run build
npm run serve
```
